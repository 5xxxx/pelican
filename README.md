# pelican

pelican 是一套基础工具集,其中包括:
- 常用加密解密方法
- 入参验证器
- (基于echo) 返回值统一包装
- (基于echo) 返回值错误处理

## Installation

Install my-project with go get

```bash
  go get  github.com/5xxxx/pelican
```
